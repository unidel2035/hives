# Простой Node.js сервер

## Установка и запуск

1. Перейдите в папку:
```bash
cd simple-server
```

2. Запустите сервер:
```bash
npm start
# или
node server.js
```

3. Откройте в браузере:
- http://127.0.0.1:3000/ - главная страница
- http://127.0.0.1:3000/api - API endpoint

## Остановка сервера

Нажмите `Ctrl + C` в терминале
